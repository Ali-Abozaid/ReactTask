## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Ali-Abozaid/ReactTask.git
```

2. Navigate to project folder:

```bash
cd ReactTask
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm start
```

---

## 🚀 Build for Production

```bash
npm run build
```
